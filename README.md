Welcome to **RECAP 2025**, an immersive data-driven visualization component for Obsidian. Designed to showcase yearly development milestones, travels, and releases, it combines a responsive canvas node graph with dynamic media preview panels to bring milestone journals to life.

---

## Quick Start

To start using RECAP 2025 today:
1. **Download the Repository**: Clone or download this repository directly into any folder inside your Obsidian vault.
2. **Install Datacore**: Ensure you have the **Datacore** plugin installed and enabled in Obsidian.
3. **Open the Entry Note**: Open the **`RECAP 2025.md`** note inside Obsidian to launch the component!

---

## Features

### Interactive 3D Node Graph
*   **Fluid Orbit Controls**: Spin, scroll, and drag the interactive 3D ring configurations representing different years and months.
*   **Procedural Node Connections**: Custom mathematical layout that renders active developer milestones as floating orbital nodes with responsive connecting fibers.
*   **Theme Synchronization**: Pulls color tokens and accent values dynamically from the host Obsidian theme to match default or community layouts.

### Media Dashboard & Auto-Play
*   **Cinematic Presentation**: An elegant, high-impact viewport that displays development recordings and website summaries.
*   **Interactive Slider**: Seamless carousel for months featuring multiple milestones or media items.
*   **Hands-Free Playback**: Turn on auto-play to automatically transition through yearly achievements cinematic-style.

### Tactical Feedback & Reactive UI
*   **Community Overlays**: Real-time styling of Discord and Reddit comments aligned with specific releases.
*   **Emoji Reactions**: Micro-animations and HUD elements mapping actual milestone sentiment.
